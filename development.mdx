---
title: "Local development and preview for FW Mining OS docs"
description: "Run the FW Mining OS documentation site locally using the Mintlify CLI for a live preview while editing or contributing to the docs."
icon: 'terminal'
---

Use the Mintlify CLI to preview the documentation site locally while you write or edit pages.

<Steps>
  <Step title="Install the Mintlify CLI">
    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Preview locally">
    Navigate to the docs directory and run:
    ```bash
    mint dev
    ```
    A local preview is available at `http://localhost:3000`.
  </Step>
  <Step title="Validate before publishing">
    Check for configuration errors and broken links:
    ```bash
    mint validate
    mint broken-links
    ```
  </Step>
</Steps>
